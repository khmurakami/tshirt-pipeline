# tshirt-airflow

# Install

```shell
$ ./download_models.bash
```

```shell
$ git pull --recurse-submodules
```

Go into every folder inside lib and run:

```shell
$ python setup.py install
```

Then in the root folder

```shell
$ python setup.py install
$ pip install -r requirements.txt
```

# Run

```shell
$ python setup.py install
```
